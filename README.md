## рҹҺҜ What Is This?

> **Upload any CSV or Excel file вҶ’ get a full BI dashboard, forecasts, and business insights in seconds.**

The **BI & Forecasting Platform** is an open-source, web-based analytics tool that lets anyone вҖ” regardless of technical background вҖ” go from raw data to actionable business insights without writing a single line of code.

Think of it as a **lightweight, free alternative to Power BI or Tableau**, with built-in data cleaning, time-series forecasting, and auto-generated business recommendations вҖ” all running locally or in the cloud.


## вңЁ Key Features

| Feature                      | Description                                                            |

## рҹ“„ Page-by-Page Guide

### рҹҸ  Home
- Displays live session KPIs: dataset rows, columns, models trained, best accuracy
- Workflow guide showing the 7-step analytics journey
- No computation вҖ” purely informational

### рҹ“Ӯ Upload Data
- Drag & drop CSV or Excel upload with multi-encoding support
- Built-in sample dataset (1,200-row synthetic sales/churn data)
- Instant preview: row count, column count, missing cells, duplicates
- Column type inspector with null percentages

### рҹ§№ Data Cleaning
The auto-cleaning pipeline runs **6 sequential steps**:

```
Step 1 вҶ’ Drop fully empty rows
Step 2 вҶ’ Impute missing values  (median for numeric, mode for categorical)
Step 3 вҶ’ Remove duplicate rows
Step 4 вҶ’ Cap outliers           (IQR method: clip at Q1-1.5Г—IQR and Q3+1.5Г—IQR)
Step 5 вҶ’ Encode categoricals    (LabelEncoder вҶ’ adds *_enc columns)
Step 6 вҶ’ Engineer date features (month, day-of-week, quarter)
```

Reports: missing values fixed, duplicates removed, outliers capped, quality score (0вҖ“100%)

### рҹ”Қ EDA Analysis
- **Distribution chart** вҖ” histogram + marginal box plot for any numeric column
- **Scatter plot** вҖ” X vs Y with optional colour-coding by category
- **Box plot** вҖ” spread comparison across segments
- **Correlation heatmap** вҖ” Pearson r matrix for all numeric features
- **Top correlations table** вҖ” ranked by absolute correlation value

### рҹ“Ҳ BI Dashboard
- **KPI cards** вҖ” sum and average for top numeric columns
- **Time-series area chart** вҖ” daily / weekly / monthly frequency toggle
- **Category bar chart** вҖ” any metric broken down by any category
- **Segment donut chart** вҖ” proportional share visualisation
- **CSV export** вҖ” download the full dataset

### рҹ’Ў Insights Report
Auto-generated findings using **6 statistical rules**:

| Rule Name       | Condition                                                                  | Priority |
| --------------- | -------------------------------------------------------------------------- | -------- |
| Skewness Check  | Any column with absolute skewness value greater than 2                     | Medium   |
| Pareto Analysis | Top 20% of rows contribute more than 60% of revenue                        | High     |
| Target Rate     | Positive rate exceeds the 20% industry benchmark                           | High     |
| Top Driver      | Feature with the highest Pearson correlation with target                   | High     |
| Segment Gap     | Best performing segment generates 1.5Г— more revenue than the worst segment | Medium   |
| Trend Detection | Change between first half and second half exceeds 10%                      | Medium   |


Each finding is colour-coded: рҹ”ҙ High &nbsp; рҹҹЎ Medium &nbsp; рҹҹў Low

## рҹҡҖ Performance & Caching

The platform uses Streamlit's caching system so heavy operations run **only once**:

```python
@st.cache_data          # For data (DataFrames, dicts, lists)
def clean_dataframe(df): ...

@st.cache_data
def run_forecast(_df, date_col, value_col, ...): ...

@st.cache_resource      # For objects (ML models, pipelines)
def train_all_classifiers(_df, feature_cols, target): ...
```

| Function                   | Cache Type   | Purpose                     | Performance Benefit                |
| -------------------------- | ------------ | --------------------------- | ---------------------------------- |
| `generate_sample_data()`   | `cache_data` | Generates demo dataset      | Runs only once per session         |
| `clean_dataframe()`        | `cache_data` | Data cleaning pipeline      | Prevents repeated cleaning         |
| `get_summary_stats()`      | `cache_data` | Computes dataset statistics | Instant results for large datasets |
| `get_correlation_matrix()` | `cache_data` | Correlation calculations    | Saves ~2 seconds                   |
| `run_forecast()`           | `cache_data` | Time-series forecasting     | Avoids repeated ARIMA training     |
| `generate_insights()`      | `cache_data` | Insight generation          | Faster report creation             |

> **Result:** After the first load, switching between pages is near-instant.

MA(1) вҶ’ Corrects for past prediction errors
